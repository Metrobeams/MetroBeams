# Plano: Módulo de Categorias de Ativos

## [S1] Objetivo

Implementar o cadastro de categorias de ativos por organização, seguindo os padrões existentes do projeto:
- Isolamento multi-tenant (organization_id)
- Autorização no servidor (Bodyguard Policy)
- TDD obrigatório
- Phoenix Controller + HEEx + Tailwind

## [S2] Escopo

### Funcionalidades
1. **Schema de Categorias** (`asset_categories`)
   - Campos: `id`, `organization_id`, `name`, `description`, `inserted_at`, `updated_at`
   - Índice único: `organization_id + name` (mesmo nome em orgs diferentes é permitido)
   - Soft delete via `active` boolean

2. **Contexto `Assets`** (`lib/plataforma/assets.ex`)
   - `list_categories/2` - Listar categorias da organização
   - `get_category!/2` - Buscar categoria por ID (com isolamento)
   - `create_category/3` - Criar categoria (com autorização)
   - `update_category/3` - Editar categoria (com autorização)
   - `delete_category/2` - Desativar categoria (soft delete)
   - `change_category/2` - Changeset para formulários

3. **Controller** (`lib/plataforma_web/controllers/asset_category_controller.ex`)
   - `index` - Listar
   - `new` - Formulário de criação
   - `create` - Criar
   - `edit` - Formulário de edição
   - `update` - Editar
   - `delete` - Desativar

4. **Templates HEEx** (`lib/plataforma_web/controllers/asset_category_html/`)
   - `index.html.heex`
   - `new.html.heex`
   - `edit.html.heex`
   - `form.html.heex` (componente compartilhado)

5. **Policy** (`lib/plataforma/assets/policy.ex`)
   - `manage_categories` - owner, admin, technician

6. **Testes**
   - Context: `test/plataforma/assets_test.exs`
   - Controller: `test/plataforma_web/controllers/asset_category_controller_test.exs`
   - Tenant isolation: `test/plataforma/assets/tenant_isolation_test.exs`

## [S5] Migration

```elixir
create table(:asset_categories, primary_key: false) do
  add :id, :binary_id, primary_key: true
  add :organization_id, references(:organizations, type: :binary_id, on_delete: :delete_all), null: false
  add :name, :string, null: false
  add :description, :string
  add :active, :boolean, null: false, default: true
  timestamps(type: :utc_datetime_usec)
end

create unique_index(:asset_categories, [:organization_id, :name],
         name: :asset_categories_organization_name_unique_index)
create index(:asset_categories, [:organization_id, :active])
```
